Search Help:

  • Wildcards: *
  • Keyword Prescence: Default is OR multiple words. Can use + say term must be present or - to say term must not be present. `+foo bar -baz` must contain foo, might contain bar and must not contain baz
  • Dashes in Term: if you use a dash as part of the term, it must be escaped with a `\` e.g. `aspnet\-core'
  • Search Fields: you can narrow the search by looking in specific fields. title, subtitle, categories,description
